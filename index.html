Wanna Trade Cards?

Skills Used: C#.NET, SQL, jQuery, AngularJS, HTML5, CSS, JavaScript, Bootstrap

Check it out!

A mobile-first, single-page web app created to help trading card game players quickly check the value of their cards when trading with other players to make sure it's a fair deal. The site features a tool to add cards to a trade and see the total value offered by both parties. It also provides an easy to nagivate browsing module to look up individual cards, see their details, value, etc. Currently supporting Pokemon, Magic the Gathering and Yu-Gi-Oh.
